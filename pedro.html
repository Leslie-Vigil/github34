Mis Aprendizajes del Año

Creación de Formularios en NetBeans

A lo largo de este año, aprendí a crear formularios en NetBeans y a conectarlos con bases de datos en SQL. Esto me ha permitido entender cómo capturar y manejar datos desde una aplicación en Java, conectándola de manera efectiva a una base de datos.

Creación y Análisis de Bases de Datos en SQL

He trabajado en la creación de bases de datos, el análisis de los problemas y los requisitos para estructurarlas de manera adecuada. Esto me ha dado una mejor comprensión sobre la organización y gestión de datos en SQL.

HTML y JavaScript Avanzado

Recientemente, he estado aprendiendo HTML de manera avanzada, explorando funciones en JavaScript, el uso de ventanas emergentes para capturar datos, y otras instrucciones como alert para interactuar con el usuario de manera básica.
