Estilo css


Color y Fuente: Elegí un fondo negro y un color blanco para el texto, generando
un alto contraste que facilita la lectura y da una sensación de elegancia.
La fuente sans-serif asegura una apariencia moderna y limpia.
Estructura y Espaciado: Usé padding y margin en los campos y en el contenedor
para que todo tenga espacio suficiente y no se vea amontonado.
Redondeé los bordes con border-radius para hacer el diseño más suave.
Botón de Acción: Al botón de cálculo le di un color verde para destacarlo, ya
que es el elemento más importante para el usuario.
Con cursor: pointer, hice que el cursor cambie al pasar sobre el botón,
indicando que es interactivo.

El diseño busca no solo hacer que la página se vea bien, sino también que el usuario
entienda de inmediato cómo completar el formulario y obtener su cotización. En pocas
palabras, esta página facilita el proceso de reserva, evitando pasos adicionales y
permitiendo una experiencia de usuario rápida y amigable
